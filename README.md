# OCR GCSE (J277) Unit 2 Revision Helper
A cool website tailored for the OCR J277 spec for computer science I have added as much information from the textbook as possible and hopefully this helps someone with their exams.
## Features
**Interactive Binary Tools:**
* **Live Text to Binary Translator:** Type in text and instantly see it converted to binary.
* **Binary Flipper:** Click individual bits (128, 64, 32...) to see how they impact the total denary value.
* **Binary Shifts:** Input a denary value and visually perform logical left and right shifts.
* **Test Your Knowledge:** A quiz section that randomly generates image and sound file size calculation questions so you can test your skills.
